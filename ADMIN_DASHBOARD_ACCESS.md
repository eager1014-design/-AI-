## 🔍 자주 묻는 질문 (FAQ)

### Q1. 관리자 페이지가 안 보여요!
**A**: 다음을 확인하세요:
1. 올바른 이메일로 로그인했는지 확인 (`eager1014@gmail.com`)
2. 비밀번호가 `ea787878`인지 확인 (admin1234 ❌)
3. 브라우저 캐시 삭제 후 재시도
4. 관리자 로그인 페이지 사용: `/admin-login.html`

### Q2. "관리자 권한이 필요합니다" 에러
**A**:
- 일반 사용자 계정으로 로그인한 경우입니다
- 관리자 계정 (`eager1014@gmail.com`)으로 다시 로그인하세요

### Q3. 정산 계좌 정보가 안 보여요
**A**:
- 대시보드 탭 (기본 화면)에 있는지 확인
- 페이지 최상단 금색 박스를 확인하세요
- F5 키로 새로고침해보세요

### Q4. 비밀번호를 잊어버렸어요
**A**:
- 로그인 페이지에서 "비밀번호를 잊으셨나요?" 클릭
- 이메일 + 전화번호로 본인 인증
- 새 비밀번호 설정 (15분 유효)

### Q5. 통계가 0으로 나와요
**A**:
- 아직 사용자나 구매가 없는 경우입니다
- 테스트 계정을 만들어 구매해보세요
- 실제 사용자가 생기면 자동으로 통계가 업데이트됩니다

## 🛠️ 기술 정보

### API 엔드포인트
```
GET  /api/admin/dashboard    - 대시보드 통계
GET  /api/admin/users        - 사용자 목록
GET  /api/admin/users/:id    - 사용자 상세
GET  /api/admin/purchases    - 구매 내역
GET  /api/admin/payments     - 결제 내역
POST /api/login              - 로그인
GET  /api/user/info          - 사용자 정보
```

### 인증 방식
- JWT (JSON Web Token) 기반
- 토큰 만료: 7일
- 헤더 형식: `Authorization: Bearer {token}`

### 브라우저 저장소
- `localStorage.auth_token`: JWT 토큰
- `localStorage.user_info`: 사용자 정보
