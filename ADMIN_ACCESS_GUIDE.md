#### 📈 실시간 통계
- **전체 사용자**: 가입한 총 사용자 수
- **회원**: 회원 등급 사용자 수 (50% 할인)

- **게시글**: 커뮤니티 게시글 수
- **댓글**: 커뮤니티 댓글 수
- **완료된 결제**: 성공적으로 완료된 결제 건수
- **주간 신규 가입**: 최근 7일 신규 가입자 수
- **주간 신규 구매**: 최근 7일 구매 건수

- 프롬프트별 판매 수량
- 프롬프트별 총 매출

## ❓ 자주 묻는 질문

### Q1. 관리자 사이트에 접속할 수 없습니다
**A**: 다음을 확인하세요:
1. 메인 페이지에서 먼저 로그인했는지 확인
2. 올바른 관리자 계정 (`eager1014@gmail.com`)으로 로그인했는지 확인
3. 브라우저 쿠키/로컬스토리지가 활성화되어 있는지 확인
4. 로그인 후 자동 리다이렉트가 작동하지 않으면 수동으로 대시보드 URL 접속

### Q2. "관리자 권한이 필요합니다" 메시지가 나옵니다
**A**: 
- 일반 사용자 계정으로 로그인한 경우입니다
- `eager1014@gmail.com` 계정으로 다시 로그인하세요
- 다른 관리자를 추가하려면 데이터베이스에서 `is_admin` 필드를 `true`로 설정해야 합니다

### Q3. 정산 계좌 정보가 보이지 않습니다
**A**:
- 대시보드 탭의 최상단에 금색 박스로 표시됩니다
- 페이지를 새로고침(F5)해보세요
- API 로딩 오류가 있는지 브라우저 콘솔(F12)을 확인하세요

### Q4. 로그인 토큰이 만료되었습니다
**A**:
- JWT 토큰은 30일 후 자동 만료됩니다
- 로그아웃 후 다시 로그인하세요
- 보안을 위해 정기적으로 재로그인하는 것을 권장합니다

### Q5. 비밀번호를 변경하고 싶습니다
**A**:
- 현재는 데이터베이스에서 직접 변경해야 합니다
- 향후 업데이트에서 비밀번호 변경 기능이 추가될 예정입니다

### Q6. 결제가 들어왔는데 정산 계좌로 입금이 안됩니다
**A**:
- 표시된 계좌는 **정산 수령 계좌 정보**입니다
- 실제 PG사(토스/카카오/PortOne) 연동 후 자동 정산됩니다
- PG사 가입 및 계좌 등록은 `PAYMENT_GUIDE.md` 참조

## 🔧 기술 정보

### API 엔드포인트
- `POST /api/login` - 로그인
- `GET /api/user/info` - 사용자 정보 조회
- `GET /api/admin/check` - 관리자 권한 확인
- `GET /api/admin/dashboard` - 대시보드 통계
- `GET /api/admin/users` - 전체 사용자 목록
- `GET /api/admin/users/{id}` - 특정 사용자 상세
- `GET /api/admin/purchases` - 구매 내역
- `GET /api/admin/payments` - 결제 내역

### 인증 방식
- JWT (JSON Web Token) 기반 인증
- 토큰 만료 기간: 30일
- 헤더 형식: `Authorization: Bearer {token}`

### 데이터베이스
- SQLite3: `instance/jjinbubu_market.db`
- 주요 테이블: `user`, `purchase`, `payment`, `post`, `comment`
