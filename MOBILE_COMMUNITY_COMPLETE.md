## 📊 기술 스택

### Backend (Flask)
- **SomoimPost 모델**: 소모임 게시글 전용 데이터베이스
- **API 엔드포인트**: 
  - `GET /api/somoim/posts` - 게시글 목록
  - `GET /api/somoim/posts/<id>` - 게시글 상세
  - `POST /api/somoim/posts` - 게시글 등록 (관리자)
  - `PUT /api/somoim/posts/<id>` - 게시글 수정 (관리자)
  - `DELETE /api/somoim/posts/<id>` - 게시글 삭제 (관리자)
  - `POST /api/somoim/posts/<id>/like` - 좋아요

### Frontend (Vanilla JS + HTML/CSS)
- **community-mobile.html**: 모바일 전용 페이지
- **실시간 데이터 로딩**: Fetch API 사용
- **탭별 필터링**: 전체/소모임/공지/정모
- **반응형 디자인**: 모바일 최적화
