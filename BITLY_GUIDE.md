## 📝 2분 완성 가이드

### 1단계: bit.ly 접속 (30초)
```
https://bitly.com/
```
- "Sign up" 또는 "Get Started for Free" 클릭
- 이메일로 가입 또는 Google 계정으로 로그인

### 2단계: URL 단축 (1분)
1. 로그인 후 상단에 "Create" 버튼 클릭
2. "Enter long URL" 칸에 붙여넣기:
   ```
   https://8003-ieqskqqgv5heqsrwn2d81-18e660f9.sandbox.novita.ai
   ```
3. 하단 "Customize back-half" 클릭
4. 원하는 이름 입력:
   ```
   jjinbubu-ai
   ```
   또는
   ```
   jjinbubu
   ```
5. "Create" 버튼 클릭

### 3단계: 완료! (30초)
생성된 짧은 URL:
```
https://bit.ly/jjinbubu-ai

```

**결과 예시:**

```
https://www.rebrandly.com/
```
- 무료 플랜: 500개 링크
- 커스텀 도메인 연결 가능

**결과 예시:**

```

```
https://short.io/
```
- 클릭 통계 제공
- QR 코드 자동 생성

**결과 예시:**
```
