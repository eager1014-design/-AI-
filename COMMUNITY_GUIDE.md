## 📊 게시글 관리

### 게시글 목록 보기
커뮤니티 탭에서 모든 게시글을 테이블 형식으로 확인할 수 있습니다:
- **ID**: 게시글 번호
- **카테고리**: 공지/질문/자유 (색상 배지)
- **제목**: 게시글 제목
- **작성일**: 작성 날짜
- **이미지**: 이미지 첨부 여부
- **액션**: 수정/삭제 버튼

### 게시글 수정
1. 수정할 게시글의 **"✏️ 수정"** 버튼 클릭
2. 수정 폼에서 내용 변경
3. "수정하기" 버튼 클릭

### 게시글 삭제
1. 삭제할 게시글의 **"🗑️ 삭제"** 버튼 클릭
2. 확인 메시지에서 "확인" 클릭

## 🔧 문제 해결

- 브라우저 캐시 삭제 후 재시도

- 몇 초 대기 후 다시 확인
