# 소모임 연동 시스템 가이드

## 📱 개요

소모임(Somoim) 플랫폼과 찐부부 AI 프롬프트 마켓의 완벽한 연동 시스템입니다. 소모임에서 올라오는 공지사항, 정모 안내 등이 자동으로 메인 홈페이지 커뮤니티에 표시됩니다.

## ✨ 주요 기능

### 1. 자동 회원 가입
- 소모임 링크를 통해 접속한 사용자는 자동으로 회원 가입
- 회원 혜택 자동 적용 (30% 기본 할인 + 3시간 추가 50% 할인)
- 임시 계정 생성 후 정보 업데이트 가능

### 2. 게시글 자동 동기화
- 소모임 공지사항 → 메인 홈페이지 자동 표시
- 정모(정기모임) 안내 → 날짜/장소 정보 포함
- 일반 게시글 → 통합 커뮤니티에 표시

### 3. 모바일 최적화
- 터치 친화적인 UI
- 빠른 로딩 속도
- 하단 네비게이션 바
- 플로팅 글쓰기 버튼

## 🎯 게시글 유형

### 1. 공지사항 (notice)
- 중요한 알림 및 공지
- 금색 배지로 표시
- 예시: "회원 할인 이벤트 진행 중!"

### 2. 정모 안내 (meeting)
- 오프라인 모임 일정
- 날짜와 장소 정보 포함
- 파란색 배지로 표시
- 예시: "12월 정모 안내 - 강남역 카페"

### 3. 일반 게시글 (general)
- 자유로운 정보 공유
- 질문 및 답변
- 노하우 공유

## 🔧 API 엔드포인트

### 소모임 게시글 조회
```
GET /api/somoim/posts
Parameters:
  - type: all | notice | meeting | general
  - page: 페이지 번호 (기본 1)
  - per_page: 페이지당 게시글 수 (기본 20)
```

### 소모임 게시글 상세
```
GET /api/somoim/posts/<post_id>
```

### 소모임 게시글 등록 (관리자 전용)
```
POST /api/somoim/posts
Headers:
  Authorization: Bearer <admin_token>
Body:
  {
    "somoim_id": "8cafe332-cbff-11ef-b613-0a50aa12fbb11",
    "post_type": "notice | meeting | general",
    "title": "게시글 제목",
    "content": "게시글 내용",
    "author": "작성자 이름",
    "meeting_date": "2025-12-18T19:00:00",  // 정모인 경우
    "meeting_location": "장소 정보"          // 정모인 경우
  }
```

### 소모임 게시글 수정 (관리자 전용)
```
PUT /api/somoim/posts/<post_id>
Headers:
  Authorization: Bearer <admin_token>
```

### 소모임 게시글 삭제 (관리자 전용)
```
DELETE /api/somoim/posts/<post_id>
Headers:
  Authorization: Bearer <admin_token>
```

### 좋아요
```
POST /api/somoim/posts/<post_id>/like
```

## 💡 사용 시나리오

### 시나리오 1: 소모임 회원이 처음 접속
1. 소모임 링크 클릭 → 메인 페이지 접속
2. 자동 회원 가입 모달 표시
3. 회원 혜택 안내 (50% 추가 할인)
4. 자동 로그인 완료

### 시나리오 2: 정모 공지 작성
1. 관리자 로그인
2. 관리자 대시보드 접속
3. 소모임 게시글 작성 (정모 유형 선택)
4. 날짜, 장소, 내용 입력
5. 등록 → 즉시 모바일 커뮤니티에 표시

### 시나리오 3: 모바일에서 게시글 확인
1. 모바일로 커뮤니티 접속
2. 탭에서 "정모" 선택
3. 정모 게시글 확인 (날짜/장소 정보 포함)
4. 댓글로 참석 의사 표시

## 📊 데이터베이스 구조

### SomoimPost 모델
```python
class SomoimPost(db.Model):
    id                # 게시글 ID
    somoim_id         # 소모임 고유 ID
    somoim_post_id    # 소모임 측 게시글 ID
    post_type         # notice | meeting | general
    title             # 제목
    content           # 내용
    author            # 작성자
    meeting_date      # 정모 일시
    meeting_location  # 정모 장소
    is_synced_to_main # 메인 홈페이지 표시 여부
    views             # 조회수
    likes             # 좋아요 수
    created_at        # 생성일시
    updated_at        # 수정일시
```
